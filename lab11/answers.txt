What is the big O runtime of insert on a set? A map?
    log(n) for both. 
What is the difference between a set and a map?
    set is a set of keys, where a map is a set of keys to values. 
Why are sets and maps useful?
    maps are useful for Lookup tables and other key/value pair instances. sets are useful when you want to group things into categories.
